一、品牌升级后，有哪些改变?

新品牌

EventBank捷会易现品牌全新升级，并更名为 Glue Up。自2013年以来，EventBank捷会易一直专注于为客户提供专业的一体化云解决方案，帮助客户改善和提升关系互动管理。全新牌名称Glue Up，承载了我们一体化云解决方案的亮点优势，体现了产品的成长，以及我们将您的企业或社群互动发展作为使命。

我们的一体化云平台为全球50多个国家的企业和组织机构提供了强有力的服务支持，旨在帮助他们提升与联系人或会员间的互动关系，实现长期商业价值，并使互动更高效。

作为便捷易用的SaaS平台，我们是行业领先的云解决方案提供商，为企业、活动组织者、商会、协会、非营利组织、代理机构、营销人员等提供一套完整的一体化云端管理工具，旨在帮助他们实现简化运营流程，全面实现云端自动化、智能化，并消除原先管理中的困难与挑战。

关于新品牌Glup Up的Logo

我们的新Logo更大方，更简洁，更美观。它由三个圆点组成，其中两个相互连接，它代表了我们的一体化云解决方案与我们的客户之间相互连接的紧密关系。

我们希望向我们的客户（企业和组织机构等）传递关系互动的重要性，因为这是Glue Up的重要使命。

新Logo将在系统平台和移动端App中更新。我们的系统电子邮件也将采用新Logo。

我们的团队由来自全球20多个国家的专业人士组成，致力于为每一位客户提供最高水平的创新技术和服务。我们全球客户成功团队精诚合作，为您提供一流的服务和支持，协助客户成功。当然，我们将不断以高度专业的水准提高产品，增加产品功能。

三、对企业客户而言，会受到哪些影响？

这将对我的企业有何影响？

所有活动页面，营销邮件和其他链接都将自动重新定向至新域名中的对应页面，以确保服务不会中断。 如果用户尝试使用以前的EventBank.com链接访问服务，将会自动跳转至Glue Up上的新页面。

如果您之前使用EventBank API，请查看有关如何更改它的说明。

对我已发布的活动有何影响？

所有已发布的活动将自动重新定向至新链接。 即使您已经在您的官方网站，第三方网站或社交媒体上宣传了活动，也无需修改任何链接。

如果在我的网站上链接了EventBank捷会易的某些页面，会有何变化？

您的访客将被自动重新定向至Glue Up上的相应页面。 如果您愿意，也可以更新链接，但这不是必需步骤。

如果我已集成其他应用程序（例如Xero，Quickbooks，Zoom或Google Analytics）有何影响？

如果您之前已将EventBank捷会易帐户与Xero，Quickbooks，Zoom或Google Analytics进行集成，则无需采取任何措施，就可以继续使用这些集成，无需进行任何更改。

如果我使用EventBank域名作为发件人，有何影响？

如您已使用EventBank 作为发件人域名（例如abc.xyz@companyname.eventbankmail.com） 目前，它将保持不变，我们将逐步向新域名转移。 如有任何更新，我们会及时通知您。 如您新注册 Glue Up 作为发件人域名，将应用我们的新域名：abc@yourcompanyname.glueupmail.com, 此更改将于2020年9月7日生效。

EventBank 捷会易管理者 App 会有何变化？

EventBank 捷会易管理者移动端App更新为“Glue Up 管理者App”。 如果您之前下载过EventBank 捷会易管理者App，则无需再次安装或下载任何内容，只需从App Store或Google Play更新EventBank捷会易 管理者App。 重新登录，选择相应的服务器，即可继续使用管理者App。

在App Store下载Glue Up Manager 在GooglePlay下载Glue Up Manager 在豌豆荚下载Glue Up Manager
我与EventBank捷会易的合同是否会有变化？

您与EventBank捷会易签订的协议不会更改，因为我们的法人实体保持不变。

EventBank捷会易是否已被另一家公司收购？

EventBank 捷会易并未被另一家公司收购。

如何更新EventBank API？

如果您已通过EventBank API将我们平台与您的网站集成在一起，请确保将端点从EventBank更新为Glue Up（页面将重定向至 .COM / .CN / .RU上的相应服务器）。

请确保在我们的新品牌发布后的下个月更新端点，因为端点的临时重新定向将在该日期之后终止。

如果您当前正在使用我们的API，请确保在2020年9月30日之前使用新的Glue Up API更新端点

查看新的Glue Up API文档。.

以下指南可帮助您迁移至新的Glue Up API：

• API文件的EventBank.php中，将变量* $ API_endpoint *从“ eventbank.com/cn/ru”更新为“ glueup.cn/cn/ru”。

• 在将用户从您的网站重定向到EventBank的任何代码中，将包含“ eventbank.com/cn/ru”的URL更新为“ glueup.cn/cn/ru”。

• 在网站嵌入部件代码中，将包含“ eventbank.com/cn/ru”的URL更新为“ glueup.cn/cn/ru（_http://glueup.cn/cn/ru_ (http://glueup.cn/cn/ru)）”。

如果您需要任何帮助，请与我们联系 或发送电子邮件至support@glueup.cn或support@eventbank.com

我应该告诉会员/注册参会者/其他利益相关者什么信息？

2020年8月24日，您的会员，活动参与者和所有其他注册用户将退出EventBank，并自动转至Glue Up上的新登录页面，用户可以使用在EventBank捷会易上使用的登录信息，在此处进行登录。 重新定向将是自动的，因此您无需执行任何操作。
